Change fromJson (dynamic json)

Converts JSON into a Change.

Implementation 

static Change fromJson(dynamic json) {
  return Change(
    json['form'],
    json['lesson'],
    json['subject'],
    json['teacher'],
    json['room'],
    json['info']
  );
}
